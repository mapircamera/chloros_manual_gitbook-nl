# Uitvoerbeeldformaten

### TIFF (16-bits)

16-bits (digitaal pixelgetal 0 - 65535) TIFF (.tif) formaat uitvoer.

### TIFF (32-bits, procent %)

32-bits (pixel float 0,0 - 1,0) TIFF (.tif) formaat uitvoer. Pixelwaarden vertegenwoordigen reflectiepercentages waarbij 0,0 = 0% en 1,0 = 100%.

### **JPG (8-bits)**

8-bits (pixel digitaal getal 0 -255) JPG (.jpg) formaat uitvoer.

### **PNG (8-bit)**

8-bit (pixel digitaal getal 0 - 255) PNG (.png) formaat uitvoer. Verliesvrije compressie geschikt voor weergave op het web en visualisatie.
